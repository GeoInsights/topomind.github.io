# TopoMind Website

Static company website and product page for TopoMind.

## Pages

- `index.html` - TopoMind company website
- `poemai.html` - Shi Yu Shan He / PoemAI product page

## Assets

- `assets/css/home.css` - company homepage styles
- `assets/css/poemai.css` - product page styles
- `assets/js/poemai.js` - product page animation and section navigation
- `assets/images/` - brand images, product screenshots, team photos and page illustrations

## Deployment

Deploy the repository root as a static Cloudflare Pages project.

- Build command: none
- Build output directory: `.`

The site is static-only. Contact links use `mailto:info@topomind.hk`, and the PoemAI system entry points directly to `https://app.topomind.hk/login.html`.
